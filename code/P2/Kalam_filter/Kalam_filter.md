### <center> 本文档是关于卡尔曼滤波器的一些笔记

* 核心思想：本次估计值 = 上一次的估计值 + K×（本次的测量值-上一次的估计值）
* 卡尔曼增益推导：
    * 状态空间表达式：</br>
            $x_k = Ax_{k-1}+Bu_{k-1}+\omega_{k-1}\\$
            $z_{k} = Hx_{k}+\nu_{k}$
    * 上式中，有过程噪声，符合正态分布$P(\omega)\sim(0,Q)$,
    其中，$Q$是协方差矩阵(covariance matrix),$Q = E[\omega \omega^T]$
